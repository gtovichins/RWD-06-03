# Responsive Web Development - Projeto Jiboia Amazônica

## Contexto Acadêmico
Este projeto foi desenvolvido como parte da disciplina de *Responsive Web Development* da UNIVALI. A incumbência consistiu na criação de uma interface web estática utilizando HTML, focada na apresentação estruturada de uma mensagem em forma de carta.

## Objetivo
O objetivo principal foi aplicar conceitos fundamentais de desenvolvimento web, tais como:
* **Semântica HTML:** Uso correto de tags para melhor acessibilidade e SEO.
